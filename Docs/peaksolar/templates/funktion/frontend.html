Frontend


Das Frontend wird über einen Shortcode in Divi Integriert.

add_shortcode('form_shortcode', 'my_custom_shortcode');

Alles weitere wird als HTML Code geschrieben und mit Php als

 return ob_get_clean(); 

in einen temporären Buffer geschrieben und über echo geprintet.

Die Formular-Elemente sind klassische HTML-Bausteine, der Form:

 <!-- Postleitzahl -->

       <div class="form-group">

           <label for="zipcode">Postleitzahl:</label>

           <input type="text" name="zipcode" id="zipcode" placeholder="00000" maxlength="5" required>

       </div>  

und unterhalb findet sich in dem <style> tag noch css. Also wie bei einer klassische Website auch.
